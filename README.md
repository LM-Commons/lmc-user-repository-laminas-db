# lmc-user-repository-laminas-db

Laminas Db based Lmc User Repository

Authentication middleware for Mezzio applications

## Requirements

- PHP 8.2 or 8.3 (laminas-db does not support PHP > 8.3)

## Installation

You can install the lmc-user-repository-laminas-db library with composer:

```sh
$ php composer.phar require lm-commons/lmc-user-repository-laminas-db
```
